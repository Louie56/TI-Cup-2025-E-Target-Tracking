## 📖 项目简介
本系统采用上位机（Maixcam）进行图像处理，下位机（MSPM0G3507）进行运动控制的架构。通过串口通信实现数据解耦，确保了视觉追踪的实时性与云台控制的精确性。
但是这里放出的是k230的代码，过程中发现k230性能落后，遂改用maxicam，实际比赛中使用的是maxicam。本人比赛中主要负责下位机运动控制。

## 🛠️ 硬件方案
* **核心控制器**：TI MSPM0G3507 单片机 。
* **视觉处理器**：Maixcam摄像头模块 。
* **瞄准执行机构**：二维步进电机云台，搭载蓝紫激光笔 。
* **底盘驱动**：两轮差速驱动 + TB6612FNG 驱动模块 。
* **传感器**：多路模拟量灰度传感器阵列，用于黑线循迹 。

## 🚀 关键技术实现
### 1. 视觉算法逻辑 (`拟合框取点.py`)
* **多阈值颜色过滤**：使用 LAB 颜色空间对目标进行提取。
* **目标特征检测**：
    * 通过 `find_blobs` 获取色块，结合面积与长宽比过滤干扰。
    * **环状区域识别**：计算原矩形面积与中心白色色块面积之差，准确锁定靶心。
* **拟合与路径规划**：
    * 利用 `find_rects` 获取目标拟合框。
    * **角点排序算法**：将检测到的角点按顺时针（右上、右下、左下、左上）排序，为后续瞄准提供坐标参考。
    * **边线取点**：在拟合框边线上均匀取点，支持动态画圆等高级任务。

### 2. 控制算法逻辑
* **循迹控制**：使用加权平均算法计算小车相对于黑线的偏差，并通过 PID 算法调节电机速度 。
* **通信协议**：自定义数据帧格式（帧头 `0xAA 0xAA`，帧尾 `0xFF 0xFF 0xFF`），实现像素坐标的高效传输。
